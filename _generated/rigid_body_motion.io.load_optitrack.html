rigid_body_motion.io.load_optitrack¶

rigid_body_motion.io.load_optitrack(filepath, import_format='numpy')[source]¶

Load rigid body poses from OptiTrack csv export.

Parameters
filepath: str or path-like

Path to csv file.

import_format: {“numpy”, “pandas”, “xarray”}, default “numpy”

Import format for rigid body poses. “numpy” returns a (position, orientation, timestamps) tuple for each body, “pandas” returns a DataFrame and “xarray” returns a Dataset.

Returns
data_dict: dict

Dictionary with one entry for each rigid body. See import_format for the format of each entry.
